# WannaGithub
Github 登陆器，解决无法进入 github 网页的问题

**编译环境**

VS2019

---
**使用方式**

直接运行 exe 即可。

---
**为什么能上 github**

因为大陆本来就可以上 github，但是要改一下 hosts 文件，这个工具就是代替人做了这些工作。

那这么小量的工作，直接手动做不就好了吗？

因为 ip 会变化，所以有的时候如果 github 又上不去了，又得手动去各种网站解析 ip 地址，然后
再拷贝到 hosts 里面，太麻烦了，用这个工具一下就弄好了。

---
**使用的域名解析 api**

目前用的是 ip-api.com 的 api
